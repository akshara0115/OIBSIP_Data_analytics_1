# Retail Sales Data Analysis 

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a retail sales dataset to uncover meaningful insights, trends, and patterns. The goal is to help businesses make data-driven decisions by analyzing customer behavior, product performance, and sales trends over time.

## Objectives
Understand the structure and quality of the dataset
Perform data cleaning and preprocessing
Generate descriptive statistics
Analyze sales trends using time series
Study customer demographics and purchasing behavior
Visualize insights using graphs and charts
Provide actionable business recommendations

## Dataset
The dataset contains information such as:
Transaction Date
Customer ID
Gender
Age
Product Category
Quantity Purchased
Price per Unit
Total Amount

## Technologies Used
Python 
Pandas (Data Analysis)
NumPy (Numerical Computing)
Matplotlib (Data Visualization)

## Key Analysis Performed
> Data Cleaning
Handled missing values
Removed duplicates
Converted date columns into proper format
> Descriptive Statistics
Mean, Median, Mode
Standard Deviation
Summary statistics of numerical features
> Time Series Analysis
Monthly sales trends
Identification of peak sales periods
> Customer Analysis
Gender-wise spending patterns
Age distribution of customers
> Product Analysis
Revenue generated by each product category
Identification of top-performing categories
>Data Visualization
Line charts (sales trends)
Bar charts (category & gender analysis)
Histogram (age distribution)
Heatmap (feature correlation)

 ## Key Insights
Electronics and Clothing categories generate the highest revenue
Sales peak during specific months, indicating seasonal trends
Majority of customers belong to the middle-age group
Spending patterns are nearly balanced across genders
## Recommendations
Focus marketing strategies on high-performing categories
Improve sales in underperforming categories through offers
Leverage peak sales periods with promotions and campaigns
Target customers based on age and purchasing behavior
Implement loyalty programs for high-value customers
